# react-native-pencilkit

(iOS) PencilKit component for React Native

## Installation


```sh
npm install react-native-pencilkit
```


## Usage


```js
import { PencilkitView } from "react-native-pencilkit";

// ...

<PencilkitView color="tomato" />
```
